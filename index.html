Error Logs & Troubleshooting Documentation

Review standard resolution playbooks for common Windows administration and initialization edge-cases below.

Hotkeys not triggering inside specific games

Cause: Video games running with elevated privileges (like Valorant or Admin-privileged titles) block lower-level user key loops.
✔ Fix: Right-click your installed ptt.exe or desktop shortcut and select Run as Administrator.

Installer Error: "pip is not recognized"

Cause: The setup engine requires Python environmental PATH structures to execute core module injection variables.
✔ Fix: Re-run your system's Python setup wizard installer and verify that the "Add python.exe to PATH" checkbox flag is enabled.

Tkinter Error: unknown option "-line600"

Cause: Legacy build layout corruption cached by previous compiler metadata distributions.
✔ Fix: Completely wipe the local build/ and dist/ caches from the workspace and run the updated PyInstaller configuration engine script pipeline again.
